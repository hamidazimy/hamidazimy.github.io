Hansman's Taxonomy
It includes 4 dimensions:
  1. Main behavior of attack
  2. Type of target
  3. Vulnerabilities & Exploits
  4. Payload
D1: Main behavior of attack
In this dimension, attacks classify into 10 categories:
  • Virus
  • Worm
  • Trojan
  • Buffer overflow
  • Denial of service
  • Network attack
  • Physical attack
  • Password attack
  • Information gathering
  • ???
    (The book says "10 categories" but there are 9 categories listed! So...)
D2: Type of target
Targets can be Hardware or Software.
Hardware:
  • Computer equipment
    (CPU, Hard disk, etc.)
  • Network equipment
    (Router, Switch, etc.)
  • Peripheral Devices
    (Mouse, Keyboard, etc.)
Software:
  • Operating system
    (Linux, Windows, etc.)
  • Network
    (Transport layer protocol, etc.)
  • Applications
    (web server, database server, etc.)
D3: Vulnerabilities & Exploits
The CVE (Common Vulnerabilities and Exposures) project is designed to produce common definitions of vulnerabilities. Once vulnerabilities that an attack exploits are known, the relevant CVE entries can be found.
D4: Payload
Different payloads may have different effect beyond the attack itself. The payloads are classified into 5 categories:
  • First dimension attack payload
  • Corruption of information
  • Disclosure of information
  • Theft of service
  • Subversion

DoS Attacks
DoS stands for "Denial of Service"
  • Purpose: To interfere with the normal operation of network services.
  • Method: Flooding, exhausting and overwhelming the resources of the targeted network or host.
  • Resources: Network bandwidth, router’s packet forwarding capacity, name servers, memory/computing power on servers, etc.
DDoS Attacks
DDoS stands for "Distributed DoS"
  • Why? Unlike attakers, vistims are usually powerful servers with fast network connections.
  • How? The attacker distributes the attack script over multiple intermediate hosts, called zombies or bots. The attacker exploits the security, configuration, or software flaws on the zombie hosts in order to control them

Worm Attacks
  • Worms replicate themselves from system to system without the use of a host file (in spite of viruses) and perform malicious actions
  • The Morris worm is one of the first computer worms
  • The idea behind, is arget vulnerabilities in daemon processes of important network services through exploiting the static buffer overflow.
