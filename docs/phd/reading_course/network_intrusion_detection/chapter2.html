What is TCPdump
  • It's a tool used to:
    • gather data from the network
    • decipher the bits
    • display the output
  • Runs close to the metal!
TCPdump Behavior
  • By default, this reads all the traffic from the default network interface and spews all the output to the console.
  • Though, many different command-line options are available to alter the default behavior.
TCPdump Behavior
Filters
  • When you don't want to collect all the traffic.
  • example: 'tcp'
  • you can save frequently used filter to a file
    and use it by -F <filename> option.
TCPdump Behavior
Binary Collection
  • You can collect raw output in binary format for later use
  • Save the data by command tcpdump -w <filename>
  • Load the data by command tcpdump -r <filename>

Introduction to TCP
TCP is a reliable connection-oriented protocol that ensure reliability using the following mechanisms:
  • Exclusive TCP connection
    • The connection is exclusive and unique between the two hosts
  • TCP sequence number
    • These provide a sense of chronology to the TCP data sent and received.
  • Acknoledgements
    • Acknowledgements are used to inform the sender that data has been received.
